# Using Team Skills In CoCo

For full documentation, see the [README](../README.md).

This file exists as a quick-reference for team members who already have the plugin installed.

## Quick Reference

| Action | What to Type in CoCo Desktop |
| --- | --- |
| Install the repo (first time) | `I have a github which has skills in it I want to use in cortex code: https://github.com/YashEOTSS/CORTEX_SKILLS.git` |
| Sync after updates | `Sync my skills` — or click Sync in Agent Settings |
| Use a skill | `$cortex-skills:<skill-name> <your prompt>` |
| List installed skills | `/skill list` |

## Example

```text
$cortex-skills:commonwealth-data-review Help me review this dataset workflow before I share it with another team.
```
