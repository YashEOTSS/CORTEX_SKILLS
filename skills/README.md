# Skills Folder

Upload shared team skills here.

Each real skill should use this structure:

```text
skills/<skill-name>/
  SKILL.md
  LICENSE
  references/
```

Use lowercase letters and hyphens for folder names.

Good examples:

```text
skills/monthly-report-review/
skills/snowflake-cost-check/
skills/grant-data-quality/
```

Start from `_template/` when creating a new skill.

After a skill is added to GitHub, teammates should run this in CoCo:

```text
Sync my skills
```

Then use the skill:

```text
$<skill-name> <task>
```
